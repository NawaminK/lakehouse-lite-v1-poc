## Fast path

1. Read `README.md`.
2. Follow `docs/learning/onboarding-journey.md`.
3. Run `make smoke`.
4. Pick a workstream from `docs/team-roadmap.md`.
5. Use `CONTRIBUTING.md` and `docs/engineering/definition-of-done.md` before opening a PR.
