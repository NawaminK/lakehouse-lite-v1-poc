# Platform Learning Path

## Goal

Be able to start, stop, troubleshoot, and safely modify the POC.

## Learn first

1. Docker images, containers, volumes, networks, and ports.
2. Docker Compose services, dependencies, and logs.
3. Linux shell basics and file permissions.
4. Makefile commands in this repository.
5. GitHub pull request workflow.

## Hands-on tasks

- Run `make up` and `make ps`.
- Read logs for Trino, Superset, and Airflow.
- Run `make smoke`.
- Break and restore one service deliberately.
- Open one GitHub issue and submit one pull request.

## Key files

- `docker-compose.yml`
- `Makefile`
- `.env.example`
- `infra/scripts/*`
- `.github/*`
